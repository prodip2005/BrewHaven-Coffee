# ‚òï BrewHaven Coffee

**BrewHaven Coffee** is a modern, sleek coffee shop web application built using **Next.js (App Router)**. This project showcases a clean UI, dynamic routing, and an efficient in-memory backend for managing a coffee menu without the need for an external database.



---

## üìñ Project Overview

BrewHaven Coffee is designed to provide a seamless experience for both customers and administrators. Users can explore a curated coffee menu and view detailed product information, while the admin dashboard allows for easy product management.

### Key Features:
* **Dynamic Menu:** Browse a list of premium coffee items.
* **Product Details:** Individual pages for each coffee item using dynamic routing.
* **Admin Dashboard:** Add, update, or remove menu items.
* **In-Memory Backend:** High-speed data handling using Next.js API routes without database overhead.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop.
* **Instant Notifications:** Real-time feedback using React Hot Toast.

## üõ†Ô∏è Installation & Setup

To run this project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/your-username/brewhaven-coffee.git](https://github.com/your-username/brewhaven-coffee.git)
    ```
2.  **Navigate to the directory:**
    ```bash
    cd brewhaven-coffee
    ```
3.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```
4.  **Start the development server:**
    ```bash
    npm run dev
    # or
    yarn dev
    ```
5.  **Open in browser:**
    Visit `http://localhost:3000` to see the application in action.
